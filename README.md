# CluedoLite

CluedoLite is a digital simplified version of the classic investigative board game Cluedo, developed in Java with a Swing graphical interface. It supports multiple players and replicates the main deduction mechanics of the original game!

## 🎮 Features

* ✅ Support for 3–6 players
* 🃏 Automatic card distribution and secret solution generation
* 🗺️ Graphical game board with room-based movement
* 🔍 Suspicion and accusation system with automatic refutation
* 📋 Personal suspect notes table for each player
* 🏆 Automatic victory and defeat management
